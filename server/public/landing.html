Features

Everything you need to manage links at scale

From shortening your first link to running enterprise campaigns, Linkforge has every tool built in.

Link Shortening

Create short links with custom slugs (e.g., link.co/launch). Automatic slug generation, bulk link creation, link expiration dates, password protection, and redirect rules (301/302/307).

Click Analytics

Real-time click tracking with geographic breakdown (country, city), device/browser/OS detection, referrer attribution, and time-series charts. Exportable CSV reports.

Custom Domains

Connect your own domain for branded short links (e.g., go.yourcompany.com). DNS verification, SSL provisioning, and default domain fallback.

QR Code Generation

Auto-generate customizable QR codes for each link. Download as PNG/SVG, customize colors and logo overlay, track QR vs direct clicks separately.

Link Management Dashboard

Central dashboard to view, search, filter, sort, and bulk-manage all links. Tags, folders, favorites, and archive. Pagination and instant search.

Team Workspaces

Multi-user workspaces with role-based access (owner, editor, viewer). Shared link collections, activity logs, and workspace-level analytics.

API Access

RESTful API for programmatic link creation, management, and analytics retrieval. API keys with scoped permissions, rate limiting, and webhook support for real-time click event notifications.

UTM Builder

Build, save, and reuse UTM parameter templates for campaign tracking. Auto-append UTM tags to links with preset templates for Google Analytics, Meta Ads, etc.

FAQ

Questions before you commit?

Here are the answers to what teams ask most before signing up.

Can I use my own custom domain for short links?
Yes. Connect any domain you own and all your short links will use it (e.g. go.yourbrand.com/promo). Custom domains are available on the Pro and Enterprise plans.
Do links expire?
Links never expire by default. You can optionally set an expiry date or click limit on any link — after which it redirects to a custom expiry page of your choice.
What analytics data does Linkforge capture?
Every click is logged with timestamp, referring URL, country, device type, and browser. You can view this data in the dashboard, filter by date range, and export as CSV.
Is there a public API?
Yes. Linkforge offers a full REST API for creating, updating, and retrieving link data programmatically. API access is available on Pro and above.
Can I edit a link's destination after it's been shared?
Absolutely. You can change the destination URL at any time. All existing clicks will redirect to the updated destination immediately — no broken links.
Is Linkforge free to start?
Yes. The Free plan includes 500 links and 10,000 clicks per month with basic analytics. No credit card required. Upgrade any time as you grow.
